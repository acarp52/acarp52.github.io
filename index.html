Things that I've worked on

Below are some of the recent projects I've made or used extensively. This isn't a complete list, but contains some of things I'm excited about and want to show off.

ValueWatch

Python application developed to educate myself about stocks and finance, focused on discovering companies for value investing. Uses Beautiful Soup for web scraping and matplotlib for data visualization. Work in progress.

three.js

A powerful 3D library that makes using WebGL easy and intuitive. I've worked with it extensively on my current co-op, developing an augmented reality application. The feature at the top is a rotating selection of some of my favorite interactive examples of it in action.

I see you're on a mobile browser. I'm currently working through a bug that breaks this threejs feature on mobile. I apologize for the inconvenience. In the meantime, please take a look at my site on a desktop for the full experience!

GetUp

An Android app that lets you know when you've been sitting for too long. This was my first exposure to the Android SDK. I've since grown my knowledge of Android on co-op, and I have a more advanced app in development. Stay tuned!

HistoryTextPy

The goal of this project was to learn how to develop a proper language model, and to use that model to analyze a piece of text to estimate when it was written. This project gave me some great exposure to language processing and machine learning. I created it with a partner, Ben Meyers, using Python 3 and NLTK.
